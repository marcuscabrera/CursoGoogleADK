# Resumo: CursoGoogleADK

[Desenvolvendo Agentes de IA com Google ADK e Gemini 3 Pro
](https://github.com/marcuscabrera/CursoGoogleADK/wiki/Desenvolvendo-Agentes-de-IA-com-Google-ADK-e-Gemini-3-Pro)

Este repositório contém um treinamento completo sobre desenvolvimento de agentes de IA utilizando o Google Agent Development Kit (ADK) e o modelo Gemini 3 Pro. O curso está estruturado em 7 módulos progressivos, com duração total estimada de 24-32 horas.



## Estrutura do Curso:
- **Módulo 1**: [Introdução ao Google ADK e Agentes de IA (conceitos fundamentais)](https://github.com/marcuscabrera/CursoGoogleADK/wiki/M%C3%93DULO-1-%E2%80%93-Introdu%C3%A7%C3%A3o-ao-Google-ADK-e-Agentes-de-IA)
- **Módulo 2**: [Ambiente e Configuração (instalação e preparação)](https://github.com/marcuscabrera/CursoGoogleADK/wiki/M%C3%93DULO-2:-Ambiente-e-Configura%C3%A7%C3%A3o)
- **Módulo 3**: [Criando o Primeiro Agente (implementação básica)](https://github.com/marcuscabrera/CursoGoogleADK/wiki/M%C3%93DULO-3:-Criando-o-Primeiro-Agente)
- **Módulo 4**: Modelos, Ferramentas e Plugins (function calling e APIs externas)
- **Módulo 5**: Funcionalidades Avançadas (memória, callbacks)
- **Módulo 6**: Padrões de Aplicações Multiagente (arquiteturas de sistemas complexos)
- **Módulo 7**: Segurança, Deployment e Boas Práticas (produção)

## Recursos Incluídos:
- Exemplos práticos com código (agentes de atendimento, análise de conteúdo, etc.)
- Exercícios progressivos (básico ao avançado)
- Glossário técnico completo
- FAQ detalhado sobre diversos aspectos do ADK
- Recursos adicionais e links
- Cronograma sugerido em diferentes formatos (intensivo, regular, autoguiado)

O material apresenta tanto os aspectos teóricos quanto práticos do desenvolvimento com Google ADK, abordando desde a criação de agentes simples até sistemas multiagente complexos com memória persistente, ferramentas personalizadas e preparados para ambiente de produção.
